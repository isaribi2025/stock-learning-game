# Stock Learning Game

## Project Overview
The Stock Learning Game is an interactive simulation designed to teach users the fundamentals of stock trading and investment strategies. Participants can engage in a risk-free environment, where they can buy, sell, and manage virtual stocks while learning market dynamics.

## Features
- **Interactive Gameplay**: Experience real-time stock trading simulations.
- **Educational Content**: Gain knowledge through tutorials and resources on stock market basics.
- **Performance Tracking**: Monitor your progress and performance over time.
- **Multiplayer Option**: Engage with friends or other players in competitive scenarios.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT)
- **Deployment**: Heroku or Vercel for hosting

## Installation Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/isaribi2025/stock-learning-game.git
   cd stock-learning-game
   ```

2. **Install Dependencies**:
   - For the frontend:
   ```bash
   cd client
   npm install
   ```
   - For the backend:
   ```bash
   cd server
   npm install
   ```

3. **Set Up Environment Variables**:
   Create a `.env` file in the server directory and populate it with the necessary environment variables. Refer to the `.env.example` for guidance.

4. **Run the Application**:
   - Start the backend server:
   ```bash
   cd server
   npm start
   ```
   - Start the frontend:
   ```bash
   cd client
   npm start
   ```

## Usage Guide
- **Sign Up / Log In**: Create an account or log in to access the game.
- **Choose a Game Mode**: Select between single-player and multiplayer modes.
- **Start Trading**: Begin trading stocks based on your knowledge and strategies. Monitor your portfolio and make decisions accordingly.
- **Review Tutorials**: Make use of the educational resources provided to improve your trading skills.
- **Compete with Others**: Join or create challenges with friends to test your skills in a competitive framework.
